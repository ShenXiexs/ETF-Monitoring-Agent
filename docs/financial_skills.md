# 金融专家 Skills

## 目标

把通用 LLM 输出升级为“多金融专家视角编排”的研究流程，重点改善政策解析和正式研报导出。

## 当前技能库

- `产品结构分析`：回答产品定位、结构和同类位置
- `资金与成交信号`：解释成交额、净流入和持续性
- `宏观与市场影响`：映射政策或事件的市场传导路径
- `政策解读`：提炼条款变化、适用范围和执行对象
- `产品策略建议`：把研究结论翻译为产品动作或沟通抓手
- `风险与合规`：标出信息缺口、表述边界和过度解读风险
- `报告编审`：统一结构、去重并输出专业研报
- `内容叙事设计`：把研究结果转成沟通主线和内容骨架

## 模块映射

- `产品研究`：产品结构分析、产品策略建议、风险与合规
- `市场监测`：资金与成交信号、宏观与市场影响、风险与合规
- `内容策略`：内容叙事设计、产品策略建议、风险与合规
- `政策解析`：政策解读、宏观与市场影响、产品策略建议、风险与合规、报告编审

## 文档工作流

### 摘要

1. 先按 `政策解读 / 宏观与市场影响 / 风险与合规 / 报告编审` 生成技能笔记
2. 再合成为四点式政策解析摘要

### 正式研报

1. 先按 `政策解读 / 宏观与市场影响 / 产品策略建议 / 风险与合规 / 报告编审` 生成技能笔记
2. 再合成为结构化研判报告
